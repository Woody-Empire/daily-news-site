🤖 模型发布

阿里通义实验室

  • 发布 Wan2.7-Video 视频生成模型
    阿里通义实验室正式上线全新视频创作大模型 Wan2.7-Video，以"戏核驱动"为核心，支持文本、图像、视频、音频全模态输入，实现从生成、编辑、复刻、重塑到续写的全链路覆盖。模型具备局部精准修改、指令级剧情调整、最多5个主体角色锁定、多宫格故事板控制等六大核心能力，并在智能剧情设计、超40种细分表情演绎、希区柯克式变焦等复合运镜方面全面升级。目前已通过阿里云百炼平台和万相官网开放体验。
    🔗 来源

京东

  • 开源发布 JoyAI-Image 图像编辑模型权重
    京东在 Hugging Face 及 GitHub 发布统一多模态模型 JoyAI-Image 的图像编辑权重 JoyAI-Image-Edit。模型家族结合 8B 参数 MLLM 与 16B 参数 MMDiT 架构，主打理解、生成与编辑的闭环协作。JoyAI-Image-Edit 支持指令引导的精确空间操作（物体移动、旋转、相机控制）、复杂文本渲染及多视角生成。高级多图编辑及文生图版本即将发布。
    🔗 来源

OmniVoice — 由 Han Zhu、Daniel Povey 等研究人员开发的多语言语音合成项目

  • 开源覆盖600种语言的零样本TTS模型
    研究团队推出大规模多语言零样本文本转语音模型 OmniVoice，基于扩散语言模型风格的离散非自回归架构，使用58.1万小时开源数据训练，支持超600种语言，覆盖范围为同类最广。模型直接将文本映射到多码本声学Token，实时率最低达0.025。功能包括零样本克隆、属性设计、非语言声音插入及发音控制。权重、代码及论文已公开。
    🔗 来源

ElevenLabs

  • Scribe v2 升级，新增实体脱敏及关键词提示扩容
    ElevenLabs 语音转文本模型 Scribe v2 推出四项新功能：内置实体脱敏功能可自动检测并移除姓名、信用卡号等敏感信息；优化印度语系-英语语码转换，确保英语词汇以拉丁字母输出；新增 No Verbatim 模式自动剔除语气词和口吃；关键词提示容量从100个扩展至1,000个。所有功能已在 API 和 UI 全面开放。
    🔗 来源

Willow — 专注于实时语音转文本的AI公司

  • 发布 Atlas 1 语音转文本模型
    Willow 正式推出语音转文本模型 Atlas 1，基于首个专为实时听写构建的可扩展人工转录基础设施开发。官方数据显示，该模型在清晰音频和生产环境下的单词错误率分别为1.2%和2.1%，显著优于多数模型在真实环境中10%至15%的错误率。
    🔗 来源

Netflix

  • 开源首个AI模型 VOID，专注视频对象智能擦除
    Netflix 在 Hugging Face 及 GitHub 开源首个公开AI模型 VOID（Video Object and Interaction Deletion），基于 CogVideoX 架构微调，通过独特的四值掩码条件化技术，能从视频中移除指定物体并同步处理其引发的所有物理交互效果。模型包含基础推理与优化推理两阶段，要求40GB以上显存，同时提供完整的数据生成与训练代码。
